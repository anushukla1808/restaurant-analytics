**An end-to-end analytics & consulting project based on real restaurant reviews**

This repository contains a full analysis of customer reviews across five restaurants under the P&D Group.
The project applies NLP, sentiment analysis, topic modelling, and keyword extraction to uncover operational insights, customer pain points, and strategic recommendations.

**Project Highlights**
1. Cleaned & preprocessed 200+ customer reviews
2. Built sentiment scoring using NLTK VADER
3. Extracted key themes using LDA topic modelling
4. Identified monthly sentiment trends
5. Compared restaurant performance across brands
6. Generated management-ready case study reports
7. Created word clouds and visual dashboards

**Business Problem**

The P&D Group needed to understand customer sentiment across its brands:
1. Where are complaints increasing?
2. What issues drive negative reviews?
3. What topics appear most across restaurants?
4. Which restaurants are improving or declining over time?

This analysis delivers data-driven answers.
